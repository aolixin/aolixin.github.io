static变量初始化个人理解:

其实一共就两种

要么在编译时或者加载时初始化, 这两种要在类外进行赋值 —> 静态初始化

要么就在函数里初始化, 这种就是运行时初始化, 推荐用这个, 不用在类外进行赋值 —> 动态初始化

上面的个人理解不准确, 要注意: 函数中也可以静态初始化, 初始化参数是常量这种

static 函数可以是 virtual 吗?

不能, 因为普通类成员函数是有一个 this 指针的, 但是 static 函数没有this指针
